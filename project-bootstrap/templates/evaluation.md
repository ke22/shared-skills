# Evaluation Checklist — {{PROJECT_NAME}}

How I'll know this project is on track and good. 4–6 checks, no more.

## Is it clear?
- [ ] The goal fits in one sentence.
- [ ] The final output is concrete and named.

## Is it on track?
- [ ] The first 5 tasks are real next actions, not vague intentions.
- [ ] I can start today without waiting on anything.

## Is it good?
- [ ] The output does the job it was meant to do.
- [ ] Someone else (or future-me) could understand it from the README.
- [ ] Scope stayed inside the guardrails.

## Quick gut check
> If I showed this to the person it's for, would they get it in 30 seconds?
