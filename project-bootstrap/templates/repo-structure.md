# Recommended Structure — {{PROJECT_NAME}}

Pick the shape that matches the project. Keep it shallow. Add folders only when you actually have things to put in them.

## Coding / AI-agent project
```text
{{project-name}}/
├─ README.md          # human explanation
├─ AGENTS.md          # instructions for coding agents
├─ tasks.md           # next steps
├─ prompts.md         # reusable prompts for this project
├─ src/               # code
├─ data/              # inputs (gitignored if large/private)
└─ docs/              # notes, design, decisions
```
